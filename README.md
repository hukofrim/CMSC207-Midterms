# CMSC207: Midterms
Personal web profile built using HTML, CSS, and vanila JavaScript.

## Features
* **Dynamic Content Toggling**: Uses basic JS functions to navigate between sections.
* **Responsive Design**: Uses flexbox to transition from a horizontanl layout to vertical-stacking when viewed in smaller screen sizes.

## Tech Stack
* **HTML**: Webpage structure and content.
* **CSS**: Layout, formatting, and basic animations.
* **JavaScript**: Simple interactivity for dynamic content toggles.

## File Structure
* images/: Contains 3 image files used in the webpage.
* index.html
* style.css
